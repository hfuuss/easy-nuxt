# Usage
<!-- usage -->
```sh-session
$ npm install -g easy-nuxt
$ easy-nuxt COMMAND
running command...
$ easy-nuxt (-v|--version|version)
easy-nuxt/0.0.1 darwin-x64 node-v12.18.2
$ easy-nuxt --help [COMMAND]
USAGE
  $ easy-nuxt COMMAND
...
```
